YouTube Summarizer

Get AI-powered summaries of YouTube videos

API Configuration
Get your API key from console.anthropic.com.
Get your API key from open.bigmodel.cn. Format: id.secret (e.g., 1234.abcdef123456)
Preferences
Auto-open sidebar when summary is ready
Include timestamps in summary
How It Works

1. Navigate to any YouTube video with captions/subtitles

2. Click the extension icon and select "Generate Summary"

3. The extension fetches the video transcript and uses Claude AI to create a comprehensive summary

4. View the summary in the sidebar with timestamps and key points
